# РАЗРАБОТКА МОДЕЛИ ДЛЯ РАСЧЁТА СОЦИАЛЬНО-ЭКОНОМИЧЕСКИХ ЭФФЕКТОВ РЕАЛИЗАЦИИ ПРОЕКТОВ ТЕРРИТОРИАЛЬНОГО РАЗВИТИЯ 

Работа посвящена теоретическому обоснованию и разработке модели для прогнозной оценки результатов реализации проектов территориального развития и их влияния на социально-экономическое развитие соответствующей территории.  Под результатами проектов развития территорий понимается социально-экономический эффект от их реализации, а под самой территорией - соответствующее проекту муниципальное образование. 


## Актуальность и проблема исследования
Актуальность темы исследования обусловлена системными недостатками
действующей модели градостроительного планирования. Ключевыми задачами,
поставленными на высшем уровне, являются синхронизация решений
стратегического, территориального и бюджетного планирования, комплексное
развитие территорий и их цифровая трансформация.

Проблема исследования состоит в методологическом разрыве
между территориальным проектированием и недостаточно
комплексной и формализованной оценкой динамики
показателей социально-экономического развития
муниципалитета.
То есть существующие подходы к оценке не позволяют в полной
мере связать технико-экономические показатели конкретного
проекта с прогнозной динамикой показателей социальноэкономического развития территории (демография, рынок
труда, бюджетные доходы, качество городской среды).
## Цель исследования
Общей целью диссертации является разработка модели для
прогнозной оценки влияния параметров проекта развития
территории на показатели социально-экономического развития
муниципального образования.
